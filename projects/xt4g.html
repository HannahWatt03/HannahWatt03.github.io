XT4G+: Xtreme Tag 4 Girls+

This project was conducted in my third year of university as part of the Game Development course. This project was a team-based effort to create a game that allows users to play local-based online tag. This game was developed using Godot Engnine and was coded using GDScript. My main contributions to this project were desining the user interface and creating various game assets. This includes: the UI, screen transitions, and soundtrack implementation.
